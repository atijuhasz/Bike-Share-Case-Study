# Raw Data

The raw trip data files are not stored in this repository due to file size limitations.

Source: [Cyclistic / Divvy trip data (public dataset)](https://divvy-tripdata.s3.amazonaws.com/index.html)

To reproduce this analysis:

1. Download the original monthly CSV files from the official source.
2. Place them in this folder.
3. Run the script in `analysis/01_data_combining_cleaning.R`
